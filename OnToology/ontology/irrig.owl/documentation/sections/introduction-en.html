Introduction back to ToC

IRRIG is an ontology specified for the experimentation in 2019 realised by TSCF Irstea Clermont-Ferrand. This experimentation is to evaluate an irrigation system based on the irrigation method Irrinov. This document is a specification explain the IRRIG ontology version 1.0.
