Crop Growth Deductionc back to ToC or Class ToC

IRI: http://www.w3id.org/def/irrig#CropGrowthDeduction

CropGrowthDeduction - Act of carrying out an (Observation) Procedure to estimate a State of the CropGrowth Property of the Crop FeatureOfInterest. The Observation may be performed by an inference engine. Links to a State to describe what the result is. Links to a TemporalEntity to indicate when the result is valid and for how many time.
Example
A CropGrowthDeduction is made by the Software_Agent Sensor based on human observation
is equivalent to
deduction and ( has feature of interest value feature of interest crop) and ( observed property value observable property crop growth)
has super-classes
deduction

Crop growth observationc back to ToC or Class ToC

IRI: http://www.w3id.org/def/irrig#CropGrowthObservation

CropGrowthObservation - Act of carrying out an (Observation) Procedure to estimate or calculate a value of the Crop_Stage Property of the Crop FeatureOfInterest. This act can be performed during the context acquisition or context processing phases of a context aware system. Links to a Sensor to describe what made the Observation and how; links to the Crop_Stage ObservableProperty (Property) to describe what the result is an estimate of, and to the Crop FeatureOfInterest. When the act is made during the context processing phase the entry values of the procedure should be linked to the Observation by the wasDerivedFrom property.
Example
A CropGrowthObservation that made by the CatherineRoussey Sensor (human observation or biology sensor).
is equivalent to
observation and ( has feature of interest value feature of interest crop) and ( observed property value observable property crop stage)
has super-classes
observation

Crop Growth Statec back to ToC or Class ToC

IRI: http://www.w3id.org/def/irrig#CropGrowthState

CropGrowthState - A qualitative value of the CropGrowth Property which changes over time, summarizing a set of information about that Property. To evaluate the State of the Property, some Boundaries are defined.
Example
A State V2 is linked to an individual of  CropGrowth Property dedicated to maize crop. The V2 State is reached when two leaves are fully emerged with collars visible in a maize plant. The State appearance could be evaluated by human observation or by automatic method like the Growing Degree Unit (GDU).
has super-classes
state
has members
state crop growth init ni, state crop growth r1 ni, state crop growth r1d15 ni, state crop growth r5 ni, state crop growth r5hg45 ni, state crop growth v2 ni, state crop growth v7 ni, state crop growth v7d25 ni

Irrigation actuationc back to ToC or Class ToC

IRI: http://www.w3id.org/def/irrig#IrrigationActuation

IrrigationActuation - Act of carrying out an Irrigation (Actuation) Procedure to change the State of a Moisture ActuatableProperty of a Soil FeatureOfInterest using a watering Actuator. This act is performed during the context exploitation phase of a context aware system. Links to an Actuator to describe what made the Actuation and how; links to an ActuatableProperty to describe the target that the Actuation change; links to a FeatureOfInterest to detail what that property was associated with. When the act is performed based on a Deduction, it should be linked to this Deduction by the wasInfluencedBy property. Links to the Function that is triggered by the Actuation.
Example
An IrrigationActuation is made by Watering Valve Actuator, links to Soil30cmDepth FeatureOfInterest and its Soil30cmDepthMoisture ActuatableProperty.
is equivalent to
( actuation and (( acts on property acts on property value observable property soil60cm depth moisture) and ( has feature of interest value feature of interest soil60cm depth))) or (( acts on property value observable property soil30cm depth moisture) and ( has feature of interest value feature of interest soil30cm depth))
has super-classes
actuation
